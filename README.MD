# Le rectangle – 2 minutes par essai
- Début :  Un tuteur va placer votre robot aléatoirement sur le parcours et
votre robot doit démarrer avec le 5 kHz.
  -  Le robot doit suivre la ligne dans le sens anti-horaire pour faire les mandats.
    - -5 pts si vous devez replacer le robot sur la ligne
    - -5 pts si le robot de démarre pas avec le 5 kHz (départ avec le bumper arrière)
- Le robot doit détecter la couleur et allumer la DEL correspondante.
  - 5pts pour la couleur
  - Le robot doit faire le mandat demandé avec élégance
  - 6 pts le mandat est respecté
- Couleur
  - Rouge → Faire tomber la quille placée dans un rayon de 25 cm au tour de la couleur rouge.
  - Jaune → Contourner le mur
  - Bleu → Faire la danse
  - Aucune → Passer le tro
